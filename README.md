📝 To-Do List App (Streamlit)

A simple To-Do List application built using **Python and Streamlit**.  
It helps users manage daily tasks in an interactive and user-friendly interface.

---

## 🚀 Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Real-time UI updates using session state

---

## 🛠️ Tech Stack
- Python
- Streamlit
